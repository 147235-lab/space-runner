# Space Runner

**Space Runner** is a fast-paced, single-player browser game inspired by Run 3. Navigate your character through an endless series of platforms, flip gravity to avoid falling, and see how far you can go!

## Features

* Endless runner with randomly generated platforms
* Flip gravity with the `G` key to traverse upside-down sections
* Jump with `Space` and move left/right with arrow keys or `A/D`
* Scores and best scores are saved automatically in your browser
* Fully offline-ready as a PWA (Progressive Web App)
* Installable on Chromebook or any modern browser

## Controls

| Action       | Key                        |
| ------------ | -------------------------- |
| Move Left    | `Left Arrow` or `A`        |
| Move Right   | `Right Arrow` or `D`       |
| Jump         | `Space`                    |
| Flip Gravity | `G`                        |
| Restart Game | Click the `Restart` button |

## Installation (Chromebook / PWA)

1. Save the provided `index.html` file to your device.
2. Open `index.html` in Chrome.
3. Click the **Install** button in the Chrome address bar.
4. Launch the game from your home screen or apps menu.
5. Play offline anytime.

## Game Mechanics

* Platforms move from right to left simulating forward movement.
* The player can jump and flip gravity to avoid falling.
* Colliding with the bottom or top of the canvas ends the game.
* Score increases over time as you survive longer.

## Technical Details

* Developed entirely in HTML, CSS, and JavaScript.
* Uses a canvas for rendering and dynamic animations.
* Embedded service worker for offline caching.
* All game assets are self-contained in a single HTML file.
